## Setting up

Get the stack up and running using a single command. Please check [Prerequisites](getting-started/prerequisites) for the system requirements.

```bash
curl -o docker-compose.yaml https://raw.githubusercontent.com/dograh-hq/dograh/main/docker-compose.yaml && REGISTRY=ghcr.io/dograh-hq docker compose up
```
